# UE5.7_Afterlight
UE5.7 Team Project

프로젝트: Afterlight

• ​감성적인 요약: 색 잃은 폐허에서 기억의 파편을 찾아 아이의 마음속에 잠긴 여섯 개의 자물쇠를 해제하는 여정
• ​스토리 중심: 상처 입은 아이의 마음속 세계를 배경으로 잊힌 과거와 마주하며 세상의 빛과 색을 되찾아가는 이야기
• ​게임 플레이 중심: 도심 곳곳의 '기억의 파편'을 수집해 봉인된 감정을 해제하고 무채색 세계를 생명으로 물들이는 어드벤처
